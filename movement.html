Using the accelerometer

The accelerometer is built into the micro:bit.

It can return a gesture - whether the micro:bit is facing up, facing down, being shaken etc. - or x, y, z values.

Make different things happen when you use different gestures, then work out what x, y, z values the micro:bit is reading. Think about what problems you could solve by monitoring the acceleration!
